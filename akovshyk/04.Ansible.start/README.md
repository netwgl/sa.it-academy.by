  - ansible installation
  ``` bash
  sudo yum install -y epel-release && sudo yum install -y ansible
  ```
  - user creation on vagrnt boxes
  ``` bash
  sudo useradd chasey -d /home/chasey
  ```
  - user add sudo grout 
  ``` bash
  sudo usermod -aG sudo chasey
  ```

  ``` bash
  sudo passwd chasey
  ```
